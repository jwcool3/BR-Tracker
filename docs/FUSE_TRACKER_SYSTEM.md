- Analyze all accounts

## ðŸ§® Smart Recommender Logic

### Quality Score Formula:
```javascript
function calculateFuseQualityScore(brainrots) {
  let score = 0;
  
  brainrots.forEach(br => {
    // Rarity score
    const rarityScores = {
      common: 1,
      rare: 2,
      epic: 3,
      legendary: 4,
      mythic: 5,
      brainrot_god: 6,
      og: 7,
      secret: 8
    };
    score += rarityScores[br.rarity] || 0;
    
    // Mutation multiplier
    const mutationMultipliers = {
      none: 1.0,
      gold: 1.25,
      diamond: 1.5,
      bloodmoon: 2.0,
      celestial: 4.0,
      candy: 4.0,
      lava: 6.0,
      galaxy: 6.0,
      yin_yang: 7.5,
      radioactive: 8.5,
      rainbow: 10.0
    };
    score *= mutationMultipliers[br.mutation] || 1.0;
    
    // Trait bonus
    if (br.traits && br.traits.length > 0) {
      score += br.traits.length * 0.5;
    }
  });
  
  // Categorize
  if (score >= 120) return 'very_high';
  if (score >= 80) return 'high';
  if (score >= 40) return 'medium';
  return 'low';
}
```

### Distribution Algorithm:
```
1. Get all idle accounts
2. Get all available brainrots (not in active fuses)
3. Sort brainrots by quality score
4. For each idle account:
   - Assign 4 brainrots based on strategy:
     * Maximize: Top 4 per account
     * Balance: Distribute high/low evenly
     * Disposal: Bottom 4 per account
5. Ensure no account is left without 4 brainrots
6. If insufficient brainrots, flag accounts needing transfers
```
